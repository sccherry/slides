---
title: Operators
---

- Math: `+`, `-`, `*`, `/`, `%`
  - `1 + 2 -> 3`
  - `'Hello' + 'world' -> 'Helloworld'`
  - `'1' + 2 -> '12'`
- Logical: `&&`, `||`
  - `true && false -> false`
  - `true || false -> true`
- Equality: `===`, `!==`, `==`, `!=`
  - `'1' === 1 -> false`
  - `'1' == 1 -> true`
  - `'1' !== 1 -> true`
  - `'1' != 1 -> false`
